## 💻 Sobre o projeto

Marvel Heroes é um app para listagens de alguns personagens da Marvel. O projeto trata-se de uma implementação para o desafio criado pela [DevChallenge](https://devchallenge.com.br/challenges/5edabfda693fca2236c83cc7/details).

## 🚀 Como executar o projeto
Abaixo são apresentados os passos necessários para que o App funcione.

### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
[Git](https://git-scm.com), [Flutter](https://flutter.dev/docs/get-started/install). 
Além disto é bom ter um editor para trabalhar com o código como [VSCode](https://code.visualstudio.com/)

#### 🎲 Rodando o Aplicativo

```bash

# Clone este repositório
$ git clone https://github.com/willyoliv/marvel_heroes.git

# Acesse a pasta do projeto no terminal/cmd
$ cd marvel_heroes

# Instale as dependências
$ flutter pub get

# Execute a aplicação 
$ flutter run


```
